## Capture guidelines

- Chrome / Arc at 1440×900 browser size, DevTools closed.
- Incognito window to keep MetaMask accounts out of the frame.
- macOS `⌘⇧4` for region capture; save straight into this directory.
- Strip system chrome (menu bar / dock) from the crop.
- One canonical light-on-dark frame per screen — no duplicate variants.
